Cadastro de Maquinário

Sistema desenvolvido em VB.NET com integração ao SQL Server para gerenciamento de máquinas e ordens de manutenção.

Objetivo

Permitir o cadastro de máquinas, visualização dos dados em tabela, aplicação de filtros de busca e registro de manutenções realizadas.

Tecnologias Utilizadas

- VB.NET (Windows Forms)
- SQL Server
- ADODB (conexão com banco)
- DataGridView 

Funcionalidades
- Cadastro de máquinas  
- Upload de foto da máquina  
- Listagem em tabela (DataGridView)  
- Pesquisa/filtro em tempo real  
- Cadastro de ordens de manutenção  

Banco de Dados

O banco utilizado é o SQL Server.
